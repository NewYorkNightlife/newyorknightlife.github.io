# Breadcrumb Module (Sprint 2 Step 5)

Adds a reusable visible breadcrumb trail on deeper content pages.

## Behavior
- Injected automatically by `js/main.js` on pages with paths like `/section/page.html`
- Skips homepage and section index pages
- Format: `Home > Section > Page`

## Why
- Improves UX orientation on deep pages
- Strengthens internal navigation signals for users and search engines

## Scope
Current sections mapped with clean labels:
- guides
- neighborhoods
- tools
- rankings
- categories
- visit
- tonight
- weekend
- things-to-do
